# Catdex 
**Interactive web application that allows you to search and discover different breeds of cats** 

## web app setup

**Clone repository or download repository files** 

**Install all dependencies using**
npm install

**Through the terminal, copy .env.example to .env using**
cp .env.example .env  #for macOS & Linux
copy .env.example .env #for windows

**Add your CatApi key to the .env file**

**Run development server using** 
npm run dev
